---
title: MIP-20
description: Magnus token standard extending ERC-20
---

MIP-20 (Magnus Improvement Proposal 20) is Magnus's token standard for protocol-native stablecoins. It extends ERC-20 with fee token registration and FX rate interfaces.

All ERC-20 methods work on MIP-20 tokens — existing tools, contracts, and wallets that support ERC-20 support MIP-20 without changes.

## ABI

```typescript
import { MIP20 } from '@magnus/sdk/precompiles'
// Extends ERC-20 ABI with Magnus-specific functions
```

## Standard ERC-20 functions (all supported)

- `balanceOf(address)`
- `transfer(address, uint256)`
- `approve(address, uint256)`
- `transferFrom(address, address, uint256)`
- `allowance(address, address)`
- `totalSupply()`

## MIP-20 extensions

### `fxRate()`

Returns the token's current USD exchange rate, scaled to 18 decimals. Sourced from the on-chain FX reporter registered for this token.

```typescript
const rate = await client.readContract({
  address: mVND_ADDRESS,
  abi: MIP20.abi,
  functionName: 'fxRate'
})
```

### `denomination()`

Returns the ISO 4217 currency code for the token.

```typescript
const denom = await client.readContract({
  address: mVND_ADDRESS,
  abi: MIP20.abi,
  functionName: 'denomination'
})
// 'VND'
```

## Compatibility

MIP-20 tokens are fully ERC-20 compatible. No changes required for:
- MetaMask and standard wallets
- Uniswap and ERC-20-compatible DEXes
- Standard `Transfer` and `Approval` event monitoring
- OpenZeppelin ERC-20 contracts interacting with MIP-20 tokens
