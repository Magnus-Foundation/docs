---
title: FeeManager
description: Precompile for fee token registration and FX rate queries
---

The FeeManager precompile manages the registry of tokens that can be used as gas fees on Magnus. It also exposes FX rates published by the on-chain FX reporters.

**Address:** `0x...` (fixed precompile address)

## ABI

```typescript
import { FeeManager } from '@magnus/sdk/precompiles'
// FeeManager.address -- precompile address
// FeeManager.abi -- full ABI
```

## Read functions

### `registeredTokens()`

Returns all registered fee token addresses.

```typescript
const addresses = await client.readContract({
  ...FeeManager,
  functionName: 'registeredTokens'
})
// ['0xmVND...', '0xmUSD...', '0xmEUR...']
```

### `fxRate(address token)`

Returns the current FX rate for a registered token in USD, scaled to 18 decimals.

```typescript
const rate = await client.readContract({
  ...FeeManager,
  functionName: 'fxRate',
  args: [mVND_ADDRESS]
})
// 25000000000000000000000n -- represents 25,000 VND per USD
```

### `isRegistered(address token)`

Returns `true` if the token is a valid fee token.

```typescript
const valid = await client.readContract({
  ...FeeManager,
  functionName: 'isRegistered',
  args: [mVND_ADDRESS]
})
// true
```

## FX reporters

Three FX reporters publish rates to the FeeManager on Staccato devnet:

- mVND/USD reporter: publishes Vietnamese Dong exchange rate
- mEUR/USD reporter: publishes Euro exchange rate
- mUSD reporter: always returns 1.0 (USD-pegged)

Reporters are live and updating continuously. Query `fxRate()` for the current value.
