Magnus is an EVM-compatible Layer 1 blockchain built for stablecoin payments in emerging markets.

The core idea is simple: on every existing chain, you need a separate gas token to do anything. ETH on Ethereum, TRX on Tron, SOL on Solana. Magnus removes this entirely at the protocol level. You pay transaction fees in whatever stablecoin you already hold — mVND, mUSD, USDT, USDC, or mEUR.

## Built on Commonware

Magnus is built on [Commonware](https://commonware.xyz), a modular blockchain infrastructure toolkit. Commonware provides:

- **Simplex BFT** consensus — 300ms deterministic finality
- **BLS12-381 threshold signatures** — for secure cross-chain bridges
- **Production-grade P2P networking** — no custom networking code

Magnus is a Route 66 participant chain. Route 66 is a Commonware and Coinbase joint initiative to standardize exchange, wallet, and custodian integration for specialized blockchains.
